📊 Probability Distributions

Understanding the Laplace distribution — the math behind Problem 2 of the final exam

What is a Probability Distribution?

A probability distribution tells you: how likely is each possible value of a random variable?

For a continuous distribution, we use a probability density function (PDF) p(x). It has two rules:

Two rules every PDF must satisfy

1. Non-negative: p(x) ≥ 0 for all x

2. Normalised: The area under the curve equals 1:

∫-∞+∞ p(x) dx = 1

Think of it like a histogram with infinitely thin bars — the total area is 100%.

The Laplace Distribution (Problem 2)

The exam gives you this PDF:

p(x) = A · exp( −√2 · |x − μ| / λ )

Let's unpack each part:

μ (mu) — the location parameter

The centre of the distribution — where the peak is. Like the mean of a Gaussian. When x = μ, the exponent is 0, so exp(0) = 1 — maximum probability.

λ (lambda) — the scale parameter

Controls the width/spread. Large λ → wide, flat distribution. Small λ → narrow, tall peak. The exam asks you to show variance = λ².

A — the normalisation constant

This is what Part (a) asks you to find. It's the number that makes the total area equal to 1. We compute it by requiring ∫ p(x) dx = 1.

Part (a): Finding the Normalisation Constant A

We need: ∫-∞+∞ A · exp(−√2|x−μ|/λ) dx = 1

Because of the absolute value, we split the integral at x = μ:

Split: ∫-∞μ A·exp(−√2(μ−x)/λ)dx + ∫μ+∞ A·exp(−√2(x−μ)/λ)dx
Both halves are equal by symmetry. Focus on the right half:
∫μ+∞ exp(−√2(x−μ)/λ) dx. Let u = x−μ, du = dx → ∫0+∞ exp(−√2 u/λ) du
Integrate: [−λ/(√2) · exp(−√2 u/λ)]0+∞ = 0 − (−λ/√2) = λ/√2
Total = 2 · A · λ/√2 = A · √2 · λ = 1
Therefore: A = 1/(λ√2) = √2/(2λ)
A = 1 / (λ√2)

Part (b): The Mean

The mean is E[X] = ∫ x · p(x) dx.

Shortcut: The Laplace distribution is symmetric around x = μ. For any symmetric distribution, the mean equals the centre of symmetry.

Therefore: E[X] = μ

You can verify formally by splitting at μ and showing the two halves cancel.

Part (c): Variance = λ² (Integration by Parts)

Variance = E[(X−μ)²] = ∫ (x−μ)² · p(x) dx

With substitution u = x−μ and using symmetry:

Var = 2 · A · ∫0∞ u² · exp(−√2 u/λ) du  (using symmetry, factor of 2)
Let α = √2/λ. Then: 2A · ∫0∞ u² · e−αu du
Integration by parts twice: ∫0∞ u² e−αu du = 2/α³
(Using: ∫ u^n e^{−αu} du = n!/α^{n+1})
Var = 2 · (1/λ√2) · (2/α³) = 2 · (1/λ√2) · 2 / (√2/λ)³ = 2/(λ√2) · 2λ³/(2√2) = λ²
Var(X) = λ²

Part (d): Sampling the Distribution

The exam gives you a recipe to generate samples from this distribution. Let's understand why it works.

The Inverse CDF Method (Inversion Sampling)

To sample from any distribution: generate U ~ Uniform[0,1], then compute x = CDF−1(U).

This works because if U is uniform, x = CDF−1(U) has exactly the distribution with that CDF.

The exam's recipe is a simplified version of this for the Laplace distribution:

1 u1 ~ Uniform[0,1] → determines the sign (left or right of μ)
2 s = 1 if u1 < 0.5, else s = −1 → 50% chance of positive or negative direction
3 u2 ~ Uniform[0,1] → determines how far from μ
4 x = μ + s·λ·log(u2)/√2 → inversion of the exponential CDF
Note: log(u2) is negative (since 0 < u2 ≤ 1), so s·log(u2) points in the direction of s

Why x = s·λ·log(u2)/√2 gives the right shape

The right half (x > μ) follows exp(−√2·x/λ). The CDF of this is 1 − exp(−√2·x/λ).

Inverting: x = −(λ/√2)·log(1−u). Since (1−u) and u are both uniform, we simplify to x = −(λ/√2)·log(u2).

Since log(u2) is negative and we want positive x, this gives positive distance. Multiply by s to randomly assign direction.

Part (e): Verifying Variance Numerically

Generate many samples for each λ, compute their variance with numpy, compare to λ².

✅ Code pattern for Part (e):

rng = np.random.default_rng(42)
mu = 0

for lam in [1, 3, 5]:
    u1 = rng.uniform(0, 1, 50000)
    u2 = rng.uniform(0, 1, 50000)
    s  = np.where(u1 < 0.5, 1, -1)
    x  = mu + s * lam * np.log(u2) / np.sqrt(2)

    measured_var = np.var(x)
    theoretical  = lam**2
    print(f"λ={lam}: measured var={measured_var:.3f}, λ²={theoretical}, ratio={measured_var/theoretical:.4f}")
