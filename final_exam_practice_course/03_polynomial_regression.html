From Linear to Polynomial Regression

In the exam, after PCA you have a single feature x₁ (the PC1 projection) and a target t. Plotting t vs x₁ shows a curved relationship — a polynomial fit is needed.

Why polynomial regression?

Linear regression: t = w₀ + w₁·x₁ → only straight lines

Polynomial (degree d): t = w₀ + w₁·x₁ + w₂·x₁² + ... + w_d·x₁ᵈ → curves

Key insight: It's still linear regression — linear in the weights w. We just create extra features [x, x², x³, ...] and apply the same MSE formula.

How to Choose the Degree (Problem 1c)

Look at the scatter plot of t vs x₁ and ask: what shape does the relationship have?

Visual patternSuggested degreeExample
Straight line1 (linear)t increases steadily with x₁
One bend (U or ∩ shape)2 (quadratic)t has one peak or valley
S-shape or two bends3 (cubic)t changes direction twice
More complex4+ (use residual analysis)Visual inspection + trial

The Goldilocks principle: not too low, not too high

Underfitting (too low) The model can't capture the real curve — large systematic errors

Overfitting (too high) The model memorises the training noise — wiggles between points

Just right Captures the trend without chasing noise — good predictions on new data

In the exam: The data likely follows a quadratic (degree 2) or cubic (degree 3). Start with degree 2, look at residuals, go higher if needed.

The Math: Building the Design Matrix

Φ = [[1, x₁⁽¹⁾, x₁⁽¹⁾², ..., x₁⁽¹⁾ᵈ],
[1, x₁⁽²⁾, x₁⁽²⁾², ..., x₁⁽²⁾ᵈ],
...]

Then the normal equation gives the optimal weights in one shot:

W* = (ΦᵀΦ)⁻¹ Φᵀ t

Predictions: t̂ = Φ · W*

Prediction Uncertainty (Problem 1d)

The exam asks: "Determine what is the expected uncertainty of the predictions."

This means: compute the Root Mean Square Error (RMSE) on the training data. It tells you how far off your predictions typically are.

1 Compute residuals: ε = t − t̂ = t − Φ·W*
2 Compute variance of residuals: σ² = (1/m) · εᵀε
3 RMSE = σ = √(σ²) — this is the uncertainty in the same units as t

What "uncertainty" means physically

If RMSE = 5, then your predictions are typically off by ±5 units. You can say: "The expected prediction uncertainty is ±5 (1σ), or ±10 (2σ = 95% confidence interval)."

This is called the standard error of the regression.

The Exam Code (Problem 1d)

✅ Polynomial regression + uncertainty — no sklearn

degree = 2   # chosen from visual inspection in 1(c)

# Build design matrix Φ: columns are [1, x1, x1², ..., x1^degree]
def design_matrix(x, d):
    return np.column_stack([x**i for i in range(d + 1)])

Phi = design_matrix(x1_train, degree)   # x1_train from PCA step

# Normal equation: W = (ΦᵀΦ)⁻¹ Φᵀ t
W = np.linalg.inv(Phi.T @ Phi) @ Phi.T @ t_train

# Predictions on training data
t_pred = Phi @ W

# Residuals
residuals = t_train - t_pred

# Expected uncertainty (RMSE = standard deviation of residuals)
sigma = np.sqrt(np.mean(residuals**2))
print(f"Expected prediction uncertainty: ±{sigma:.3f}")

# Plot
import matplotlib.pyplot as plt
x_line = np.linspace(x1_train.min(), x1_train.max(), 200)
Phi_line = design_matrix(x_line, degree)
t_line = Phi_line @ W

plt.scatter(x1_train, t_train, label='Training data', zorder=5)
plt.plot(x_line, t_line, 'r-', linewidth=2, label=f'Degree-{degree} fit')
plt.fill_between(x_line, t_line - sigma, t_line + sigma,
                 alpha=0.2, color='red', label=f'±1σ = ±{sigma:.2f}')
plt.xlabel('x₁ (PC1)'); plt.ylabel('t')
plt.legend(); plt.title('Polynomial regression with uncertainty band')
plt.show()
